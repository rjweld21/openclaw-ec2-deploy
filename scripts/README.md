# Scripts

Utility scripts for OpenClaw EC2 management.

## Contents
- `setup.sh` - Initial server setup and hardening
- `deploy.sh` - Application deployment script
- `backup.sh` - Session and memory backup
- `restore.sh` - Restore from backup
- `health-check.sh` - Health monitoring script
- `update.sh` - Update OpenClaw Gateway
- `troubleshoot.sh` - Debugging and diagnostics

## Usage
These scripts are called by GitHub Actions and can also be run manually for maintenance.
